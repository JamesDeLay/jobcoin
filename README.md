## Development

- `npm install` deps
- Disable web security on Chrome to avoid CORS errors
    - Use: `sudo open -n -a /Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --args --user-data-dir="/tmp/chrome_dev_test" --disable-web-security`
